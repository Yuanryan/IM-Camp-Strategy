# 投影四區競技場介面設計

日期：2026-06-21

## 目標

重新設計投影常駐主畫面，採用「四區競技場」視覺方向，在 1280×720 投影解析度下不需捲動即可同時看見排行榜、Jackpot、四區資產與獨佔資訊。

## 保留項目

- 保留現有寶石 LOGO 與「IM 大富翁：迷霧資本戰」標題。
- 保留遊戲階段、時鐘與市場事件跑馬燈。
- 保留拍賣舞台、法槌與大樂透開獎的全螢幕覆蓋動畫。
- 不加入資產地圖圖片 GUI。

## 版面

- 頁面固定為單一視窗高度，根容器使用 `100dvh` 並禁止頁面捲動。
- Header 採緊湊雙層結構：第一層品牌、階段、時鐘；第二層為固定高度事件跑馬燈。
- Header 保留全螢幕切換按鈕，並隨瀏覽器全螢幕狀態切換為「還原」。
- 主內容分成左右兩欄：
  - 左欄寬度約 23.5rem，先顯示 Jackpot，再顯示資產排行榜。
  - 右欄為 2×2 四區資產表。
- 各區獨佔隊伍與過路費直接合併至對應資產表 Header，不再保留獨立的獨佔面板。

## 排行榜

- 十隊皆可在單一畫面內看見。
- 第一名使用無接縫的 RGB 霓虹流光。
- 第二名使用金色光芒與流光。
- 第三名使用銀色金屬光。
- 第四名後使用較安靜的深色列，保留淨值比較能力。

## Jackpot

- 大樂透常駐資訊只顯示獎金池，不顯示期數與已選號碼。
- Jackpot 卡片背景放置低對比度彩球，使用緩慢漂浮動畫。
- 偏好減少動態時停止彩球位移。

## 四區資產

- 每區 Header 顯示區名、主題、獨佔狀態與過路費。
- 有獨佔時，獨佔隊伍使用皇冠標籤並放大；整區套用該區主色的呼吸光暈，並有高亮光束沿著邊框持續巡迴。
- 無獨佔時顯示「競逐中」。
- 資產名稱與價值比舊版放大，擁有隊伍維持小型標籤。
- 資產等級使用三種逐級增強的燈光：一級單點、二級雙點加強、三級三點脈衝。

## 跑馬燈

- 固定標籤與捲動軌道分離，避免文字擠壓或覆蓋「市場事件」。
- 軌道固定高度、單行顯示，左右使用漸層遮罩柔化進出場。
- 兩份相同內容維持無縫循環。
- 減少動態模式顯示單行截斷文字，不增加 Header 高度。

## 驗證

- Vitest 驗證前三名視覺層級與資產等級層級。
- 執行 TypeScript、變更檔案 ESLint 與 production build。
- 在 1280×720 瀏覽器確認 `scrollHeight <= innerHeight`。
- 截圖檢查品牌、前三名、Jackpot 彩球、獨佔光暈、資產字級與跑馬燈。
